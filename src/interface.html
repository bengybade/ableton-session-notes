Put lyrics on the timeline

Tag a lyric line with a position, then use the note menu's Send lyrics to timeline… to drop a locator or a clip per line. Any untagged line falls one bar after the line above it.

tag a linelands it at
[17] wordsbar 17
[17.3] wordsbar 17, beat 3
[1:04] words1 min 4 sec — uses the Set tempo
[=8*4] wordsbeat 32 — do math with + - * / ( )
[=bpm/2] words30 s in at any tempo — bpm is the Set tempo

Bar tags like [17] stay put through tempo changes; clock tags like [1:04] assume a steady tempo.
